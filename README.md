# Clippo - Online Clipboard Tool

Online clipboard application that lets users paste text, receive a unique token,
and retrieve that text from any device using that token.

**Link to project:** https://clippo-five.vercel.app/

# Built with:

Python, Javascript, HTML, CSS, Flask, React, SQLAlchemy

# Lessons Learned:

- Understanding how frontend and backend communicate through HTTP requests
- How Flask routes handle incoming data
- How dadta is stored and retrieved from a database
- Deploying and having a publicly accessible app
